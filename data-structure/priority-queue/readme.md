### Operations

1) Insert
2) Delete
3) Peek

### Implementations

| Approach              | Insert   | Delete   | Peek |
|-----------------------|----------|----------|------|
| Arrays (unordered)    | O(1)     | O(n)     | O(n) |
| Arrays (ordered)      | O(n)     | O(n)     | O(1) |
| Linked list (ordered) | O(n)     | O(1)     | O(1) |
| Heap                  | O(log n) | O(log n) | O(1) |
| Binary search tree    | O(log n) | O(log n) | O(1) |
